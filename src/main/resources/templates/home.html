Разработать Restful API для регистрации заявок.

Функциональные требования:
  • 1. Restful интерфейс.
  • 2. Регистрация, редактирование профиля пользователя. Поля пользователя: имя, фамилия, дата рождения, адрес проживания, логин, пароль. Пароль хранить в хешированном виде.
  • 3. Получение JWT токена при вводе логина и пароля. Хранение в токене деталей пользователя.
  • 4. Создание заявки, редактирование и удаление заявки только пользователем, который его создал, получение списка заявок (сортировка по дате создания), получение данных заявки по ID. Только авторизованный пользователь может получить доступ к заявкам. Поля заявки: заголовок, текст заявки, дата создания.
  • 5. Создание комментария к заявке. Любой зарегистрированный пользователь можеть оставить комментарий к заявке. Список комментариев должен быть вложен в ответе на запрос о получении данных заявки по ID. Только авторизованный пользователь может получить доступ к комментариям. Поля комментария: текст комментария.
Нефункциональные требования:
  • 1. Java 8, 11.
  • 2. Фреймворки, библиотеки: Spring, Mybatis/Hibernate, Swagger.
  • 3. Система хранения данных: PostgreSQL.
  • 4. Приложение должно собираться при помощи maven без установки или настройки доп. компонент.
  • 5. Архив с результатом задания должен содержать файл readme.txt с инструкцией по сборке, скриптами создания объектов в БД (если не был использован liquibase) и файлом Postman для тестирования всех методов, если не был настроен Swagger.
